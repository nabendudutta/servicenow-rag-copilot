# INC0010008

**Short Description:** Terraform state file lock not releasing after failed apply in Azure
**Priority:** 3
**State:** 6

## Description
The Terraform state file for the Azure production environment is locked and preventing any new 'terraform plan' or 'terraform apply' operations. Error: 'Error acquiring the state lock'. A previous pipeline run failed mid-apply 6 hours ago and the lock was never released. Infrastructure changes are blocked.

## Resolution
ROOT CAUSE ANALYSIS:
A Terraform apply pipeline run failed mid-execution due to an Azure API timeout while provisioning a new subnet. The Terraform process was killed abruptly by the CI runner timeout (30-minute limit), which prevented the normal state unlock procedure from running at the end of the process.

RESOLUTION STEPS TAKEN:
1. Confirmed the state lock by running: terraform plan — output showed lock ID,    lock holder info, and timestamp from 6 hours prior.
2. Noted the Lock ID from the error output (e.g., abc123-def456-...).
3. Reviewed Azure Storage Account (where remote state backend resides) to confirm    the lease status on the tfstate blob via Azure portal — showed 'Leased'.
4. Verified no active Terraform process was still running in any CI pipeline.
5. Forced-released the state lock using:
   terraform force-unlock -force <LOCK-ID>.
6. Confirmed lock was released: ran terraform plan again — no lock error.
7. Reviewed the partially applied state to identify any resources in an    inconsistent state using: terraform show.
8. Ran terraform apply -target=azurerm_subnet.prod_subnet to complete the    interrupted resource creation safely.
9. Verified full infrastructure state was consistent with the plan.
10. Updated the CI pipeline configuration to increase timeout to 60 minutes and     added a post-failure hook to run terraform force-unlock automatically on     pipeline failure with the lock ID captured from stderr.

PREVENTIVE MEASURES:
Added Sentinel policy in Terraform Cloud to alert when a state lock is held longer than 45 minutes without activity.
